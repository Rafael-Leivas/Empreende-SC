# Empreende-SC

Aplicação full-stack CRUD para gerenciamento de empreendimentos catarinenses.
Desafio técnico do processo seletivo SCTEC / IA para DEVs — SENAI/SC LAB365.

## Stack

- **Back-end:** Node.js + Express + TypeScript + Prisma ORM + SQLite
- **Front-end:** React + TypeScript + Vite + Tailwind CSS v4 + shadcn/ui

## Como rodar

```bash
npm install
npm run dev
```

O comando acima instala as dependências e sobe o servidor (porta 3000) e o client (porta 5173) simultaneamente.
